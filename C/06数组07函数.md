# 数组

**定义**

```c
int a[10]; //里面是乱七八糟的数据，甚至a[11],a[100]也能取出乱七八糟的数据并且不会报错
```

**初始化**

```c
int a[10]={0}; //数组内部全部初始化为0

int a[10]={1,2,3,4,5}; //数组为{1,2,3,4,5,0,0,0,0,0}，未舒适化的后面全部为0
```
